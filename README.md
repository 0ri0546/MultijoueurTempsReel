# Arena Dash
## A real-time multiplayer game
Platform: Windows (x64) / Engine : Unreal Engine (5.6.1)

</div>

# Setup
You will need :
 - [Unreal Engine 5.6.1](https://www.unrealengine.com/en-US/download) installed on your system
 - [Git](https://git-scm.com/install/windows) installed on your system.

 Then, run the following commands inside a terminal :
```bash
git clone https://github.com/Quest-Education-Group/par-2-atari-s2p1-12
cd par-2-atari-s2p1-12
```
Finally, open the `ArenaDash.uproject` file with Unreal Engine 5.6.1.

You should be good to go!

# Building the project
Inside Unreal Engine, go to `Platforms > Windows` and click `Package Project` after choosing the correct informations for your system.

# Run
Once the game has launched, specify the `maximum number of players` and press the `Host button` to create a game.
To join a game, press `Refresh`, then `Join`.
After that, all players need to stand on the central platform so that the host can start the game.
